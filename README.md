# Newsletter in Node.js

This project uses the Mailchimp API and Node.js to manage and distribute newsletters to subscribers. It allows users to easily create and send newsletters, track the success of their campaigns, and analyze their audience's engagement. The project utilizes the Mailchimp API to store and manage user data, making it efficient and convenient for users to manage their newsletters. With this project, users can easily stay in touch with their subscribers and keep them informed about their business or organization. It requires Node.js version 8 or higher to run.

## Installation

To install this project, follow these steps:

1. Clone the repository to your local machine:

```
git clone https://github.com/roxylius/Newsletter.git
```

2. Navigate to the project directory:

```
cd Newsletter
```

3. Install the project dependencies:

```
npm install
```

## Environment Variables

There are 2 environment variables:

1. **POSTURL** : Add Endpoint URL

2. **AUTH_KEY** : API key in Format - user : "**YOUR_API_KEY**"

## Usage

To run the project, use the following command:

```
npm start
```
